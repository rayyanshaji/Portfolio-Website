Nutrional Value Calculator

Demo Link Code

Demo Video

0:03 Displaying search list   0:07 Adding Foods   0:12 Add food to a different meal

An easy to use nutrional value calculator, to track your daily caloric intake substituted with nutrtional information about each food. We've researched existing nutritional food information websites and applications and found that the user needed to perform a couple of distractive steps in order to log foods into a meal time.

We designed our application in a way which amounts to take the least steps possible for the user to reach his goal.

Notable Features
  • Auto-complete search feature for both common and branded foods to ensure a seamless user experience.
  • Selecting serving sizes automatially changes the calorie value before logging any food into a meal time.
  • Total Calories calculated for both per-meal and calories throughout the day.
Utilization of languages, technologies, frameworks, libraries and databases
Javascript

  • Used javascript for displaying Food name, image,calories, serving units in the generated search list.

JQuery

  • Used Jquery's selectors for adding foods from the list to the food information panel

  • Wrote methods for asynchnorously changing caloric value on changing serving units, adding foods to meals, and calculation of total calorie intake

AJAX

  • Used AJAX function to send a client request to the API to retrieve the response which is processed by the browser to update the search list with food items as you type the letters for the food.

NodeJS

  • Developed Node.js script for allowing user to store weight in Mongo DB database.

Nutrionix API

  • The main source of food information data from an API of over 600,000 food items.

  • Using JSON responses from the API to retrieve specific information on each food. We can show information such as proteins, carbs, fats as per users needs.

HTML

  • Using plain HTML for the main screen layout, headers, text, images, buttons and so on.

CSS

  • Used CSS styling on the autocomplete list, and used grid an flexbox to spaced elements to correctly show food information and buttons in the right areas

A full-featured web-application which is targeted towards members working within organizations in agileorscrum teams assigned to projects. We are often surrounded by overly-complicated issue-management tools which seems overwhelming to understand and get used to. It might also require you to go through endless tutorials.

This application is ideal for users who wish to interact with a simplified user interface, an get their point accross as quickly as possible. They can initiate an issue they are facing provided the details and post them. This post will be available on a home screen and other members will be able to reply to it with suggestions on solving their issue. Users are encouraged to update the case with being resolved. They also have the option to re-open a case.

Notable features
  • Register as a new user, chose your profile image
  • Change priority of case, for e.g. from Trivial to Major. Update the status of a case if it's resolved, or Reopen a case if the same issue appears again.
  • Customizable text editor where you can paste code, change text style, add bullet points, change text indentation.
  • Reply on any case with valuable suggestions on solving the issue. You can also update your reply if you've made a mistake/typo.
  • Drop downs buttons for enclosing information on case page.
This project was done as part of a graduate course I had taken named Database Programming
